# 📈 Stock AI Analytics & Prediction System

A web-based stock analysis and AI-powered prediction platform built using Flask, Machine Learning, and Real-Time Market Data.

---

## 🚀 Executive Summary

Stock AI Analytics is an intelligent application that helps users analyze stock market data, generate predictions, and visualize trends. It provides automated reports and interactive dashboards for better investment decision-making.

---

## 💼 Business Problem

Investors often struggle with:

- Analyzing large stock datasets
- Understanding market trends
- Accessing affordable AI tools
- Making data-driven decisions

This project solves these problems by providing AI-based insights, real-time data, and automated analysis.

---

## ⚙️ Methodology

### 1. Data Collection
- CSV file upload
- Live stock data using yFinance API

### 2. Data Preprocessing
- Data cleaning
- Missing value handling
- Feature engineering

### 3. Machine Learning
- Regression-based prediction models
- Performance evaluation using:
  - R² Score
  - MAE
  - RMSE

### 4. Visualization
- Interactive charts with Plotly
- Trend and forecast graphs

### 5. Reporting
- PDF report generation
- Email report delivery
- 7-day forecast
