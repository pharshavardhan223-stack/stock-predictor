  How Training Works
1
Data Preparation
The training engine generates a numeric index array x and uses your stock price series as y. No external data needed — it trains on whatever CSV you upload.
2
Linear Regression Training
Fits a straight line through your price data using sklearn.LinearRegression. Learns the slope (daily trend) and intercept (starting value).
3
Random Forest Scoring
Trains a RandomForestRegressor alongside LR to calculate R², MAE, RMSE. The best model by R² is selected as the scoring model.
4
Save to Disk
The trained Linear Regression model is saved as models/linear_model.pkl using joblib. This eliminates the startup warning permanently.
5
Ready for Predictions
Once saved, the model loads automatically on every app restart. Go to /stock or /upload to start predicting with the trained model.
